# AdvancedSoundboardInterface
Advanced Soundboard Interface for recording, renaming mp3 and wav files, replaying, button comtomization, presets, and automated responses.

=====================================================

ADVANCED SOUNDBOARD INTERFACE - Presented by: DuncanAndFriends

=====================================================

Thank you for downloading the Advanced Soundboard Interface!
This application provides a powerful, customizable soundboard
with AI-triggered playback, automatic file renaming,
recording tools, presets, and portable design.

-----------------------------------------------------
FEATURES
-----------------------------------------------------

• 20 Custom Sound Categories
    - Assign any folder to each category
    - Buttons automatically populate from the selected folder
    - Supports .mp3 and .wav files
    - Compact, dark-mode layout

• "All Sounds" Category (Recursive)
    - Select one master folder
    - Automatically includes all audio in that folder
      AND all subfolders
    - Useful for large sound libraries

• Intelligent Voice Bot
    - Saying “what?” repeats the last played sound
    - Saying anything else triggers a random sound
      from ALL categories, ALL sounds, and recordings
    - Runs silently in the background

• Recording System
    - Press "Record" to capture microphone audio
    - Saved as MP3 automatically
    - Auto-Renamer cleans names and uses transcription
    - Creates readable filenames with spaces
    - Never overwrites—adds (2), (3)... when needed

• Auto-Rename (Background Process)
    - Watches your recordings folder for new audio
    - Automatically renames based on speech-to-text
    - No popups or interruptions

• Right-Click Recording Menu
    - Rename
    - Delete (with confirmation)
    - Open Folder

• Presets (File-Based)
    - Save Preset… lets you store a preset anywhere
    - Load Preset… imports settings from a .json file
    - Presets store:
        - Category folder assignments
        - All Sounds folder
        - Recordings folder
        - Volume, pitch, and button size

• Portable Application
    - No installation required
    - All config and recordings remain next to the .exe
    - Works from USB or external drives

-----------------------------------------------------
HOW TO USE
-----------------------------------------------------

1. Launch "soundboard_app.exe".
2. Press “Choose Folder” for any category to assign sounds.
3. Scroll down to the "All Sounds" section to choose
   a master folder that includes subfolders.
4. Click any button to play a sound.
5. Enable the Voice Bot (optional):
       - Say “what” to replay last sound
       - Say anything else to play a random sound
6. Press "Record" to start/stop a microphone recording.
7. Find recordings in the "Recordings" section.
       - Right-click to rename or delete them.
8. Use "Clear Soundboard" to remove all category folders.
       - Recordings folder is not affected.
9. Save/Load presets using the preset buttons.

-----------------------------------------------------
TROUBLESHOOTING
-----------------------------------------------------

• Ensure your microphone is enabled for the recording feature.
• Ensure you are connected to the internet for transcription.
• If voice bot doesn’t respond:
      - Restart the app
      - Make sure no other program is blocking microphone access
• If SmartScreen warns about unknown publisher:
      - Click “More info” → “Run anyway”
      (This is normal for unsigned apps.)
